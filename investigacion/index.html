"Modelo Cajas"

El modelo de cajas es la característica más importante del lenguaje CSS, este establece el diseño se las páginas web.
Las cajas de una página se crean automáticamente cada vez que se define un elemento HTML, estas cajas no se observan a simple vista por que no tienen ningún color y fondo.
"Cuando la página se sube al navegador las cajas se crean y se colocan de forma automática pero con CSS se pueden modificar las características."

Propiedades del modelo de caja:

  • Margin: margen exterior, que separa la caja del elemento de las demás.
  • Padding: margen interno o relleno, que se aplica del borde hacia adentro.
  • Background-color: para asignar un color de fondo a la caja.
  • Background-image: Para asignar una imagen de fondo a la caja mediante el valor url(‘imagen.jpg’), siendo imagen.jpg la ubicación del archivo de la imagen que queramos utilizar.
  • Border-width: para definir el grosor del borde, por ejemplo, 2px.
  • Border-style: Para especificar el tipo de línea del borde. Hay varios tipos que podemos consultar aquí, pero lo normal es usar el valor solid.
  • Border-color: Asignamos el color del borde
  • Border: con esta propiedad podemos especificar las tres anteriores en una en el mismo orden que las hemos indicado.
  • Overflow: Con esta propiedad especificamos los que ocurre cuando el contenido no cabe en la caja.

La propiedad Box-sizing permite modificar la caja, poniendo el borde alrededro de ella.
Otra propiedad seria el Content-box que modifica la caja haciendo que el borde crezca hacia adentro de esta.
